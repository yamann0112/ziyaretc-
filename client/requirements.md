## Packages
framer-motion | Animations for page transitions and modal entries
date-fns | Date formatting for Turkish locale
lucide-react | Icons for UI elements
clsx | Utility for conditional classes
tailwind-merge | Utility for merging tailwind classes

## Notes
- Tailwind config should include custom colors: gold (primary), red (destructive/accent), black (background).
- Fonts: 'Outfit' for display, 'Inter' for body.
- Backend provides `/api/stats` and `/api/visitors` endpoints.
- All text must be in Turkish.
